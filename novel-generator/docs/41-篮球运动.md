# 第41类: 篮球运动

**示例小说**: 我真的只是想打铁、篮坛教皇

**写作指导**:
- **世界观设定**: 构建以篮球运动为核心的世界，包含篮球规则、职业体系、竞技文化
- **角色刻画**: 主角通常是篮球运动员或教练，需要展现篮球天赋、竞技精神、职业素养
- **背景设定**: 设定篮球环境、职业体系、比赛规则、竞技文化等元素
- **情节发展**: 以篮球比赛、职业发展为主线，融入篮球竞技、职业成长、团队合作等元素
- **语言风格**: 运用篮球术语与竞技语言，营造紧张激烈的篮球竞技氛围
